# peoples-united-financial

Profile for **People's United Financial** in the API Evangelist network.

## Status: Defunct — Acquired by M&T Bank

People's United Financial Inc. was a Bridgeport, Connecticut-based regional
bank holding company and former Fortune 1000 constituent (rank 916).

- **Acquirer:** [M&T Bank Corporation](https://www.mtb.com/)
- **Deal value:** ~$7.6 billion (all-stock)
- **Announced:** February 22, 2021
- **Closed:** April 2, 2022
- **Integration completed:** Q3 2022
- **Branches at close:** ~403 (across CT, NY, MA, VT, NH, ME)
- **Brand status:** Retired. `peoples.com` 301-redirects to `mtb.com`.

> "On April 2, 2022, the bank merged with M&T Bank and was fully integrated
> into M&T by the third quarter of 2022."
> — [Wikipedia: People's United Financial](https://en.wikipedia.org/wiki/People%27s_United_Financial)

## API surface

**Zero documented APIs.** During its operating history, People's United did
not publish a developer portal, OpenAPI/Swagger definitions, SDKs, a GitHub
organization, or any public machine-readable interface. Banking and treasury
services were delivered through proprietary online/mobile banking, ACH/wire
rails, and vendor-mediated integrations (e.g., FIS/Jack Henry-class cores) —
none of which were exposed publicly.

Any prior or hypothetical integrations have been subsumed by M&T Bank's own
(largely partner-gated) integration channels. For active banking APIs in the
M&T footprint, refer to the `mt-bank` profile in this network if/when it
materializes.
